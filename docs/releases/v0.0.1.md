# v0.0.1

🔔 What’s New

- Initial public release of `smdu` (See My Disk Usage), a modern terminal disk usage analyser inspired by `ncdu`.
- Added interactive navigation for directories and files with keyboard-first controls.
- Added sorting by name/size and flat/tree view modes.
- Added right-side status panel with selected-item details.
- Added file type colouring, including dedicated categories for text, scripts, executables/libraries, archives, and disk images.
- Added focus timer support with quick duration cycling and in-app status.
- Added settings persistence for theme, units, hidden files, and display toggles.

🧭 UX Highlights

- Header shows current path and version for better orientation.
- Footer shows totals, key hints, and scan status context.
- Information modal (`i`) shows extended metadata, including symbolic link destination state.
- Help modal (`?`) documents keybindings directly in-app.

🐛 Fixes and Reliability

- Improved symlink handling with broken-link detection and clear unresolved state messaging.
- Improved scan resilience around access/permission issues.
- Improved adaptive layout behaviour for narrow terminal widths.

🧪 Quality

- Added/expanded unit and component tests for scanner, file list, status panel, and information modal.
- Verified TypeScript build and lint/test workflows for release readiness.
